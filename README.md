# vector_distance
ベクトル間の距離を各要素の違いをもとに定義するC言語でプログラムしたリポジトリです

## 使用方法
- パッケージを入手
```
git clone git@github.com:YuichiroHatanaka/vector_distance.git
```
- 移動
```
cd src/
```
- 下記のコマンドでコンパイル  
```
gcc irreversible.c restore.c -o out -I ../include/
```
- 実行
```
./out
```
